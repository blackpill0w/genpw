# genpw
Small password generator in C++.
```
Help for genpw (by default it generates passwords with 8 lowercase letters)
	-h:	display this help
	-len n:	specify the length
	-l:	include lowercase
	-u:	include uppercase
	-n:	include numbers
	-p:	include punctuation
```
# Build
Download the `main.cpp` file and compile.
